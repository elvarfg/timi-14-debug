Niðurstaða birtist hér...

Vísbending: Notaðu console.log(typeof tala1) til að sjá hvaða tegund talan er.

Input svæði gefa alltaf strengi, ekki tölur.

// Lausn: Umbreyta í tölu
const tala = parseInt(input.value);
// eða

Æfing 3: Listi sem eyðir ekki Miðlungs

Vandamál: Þegar þú smellir á "Eyða" takkann, hverfur rangt item eða ekkert gerist.

Verkefni: Skoðaðu fallið eydaItem() og finndu villuna.

  • Epli
  • Banani
  • Appelsína
Smelltu á Eyða til að fjarlægja item

Vísbending: Vandamálið er að index breytist þegar items er eytt.

Betri lausn er að nota this til að finna rétt element:

// Í HTML:
onclick="eydaItem(this)"

// Í JavaScript:
function eydaItem(button) {
  button.parentElement.remove();
}

Æfing 7: Formið sem sendir ekki Erfitt

Vandamál: Þegar þú fyllir út formið og smellir á "Skrá" ætti skilaboð að birtast, en síðan refreshast í staðinn.

Verkefni: Þetta er klassískt vandamál! Finndu hvernig á að stöðva default hegðun forms.

Skráning birtist hér...

Vísbending: Form elements hafa "default behavior" - þau refresha síðuna.

Þú þarft að stöðva það með event.preventDefault()

function sendaForm(event) {
  event.preventDefault(); // Stöðva refresh
  // Restin af kóðanum...
}

Notendur birtast hér...

Vísbending: Athugaðu hvort fallið er async og hvort await er notað rétt.
